# Specialized queries

OpenSearch supports the following specialized queries:

- `distance_feature`
- [`hybrid`]({{site.url}}{{site.baseurl}}/query-dsl/specialized/hybrid/)
- `more_like_this`
- `percolate`
- `rank_feature`
- `script`
- `script_score`
- `wrapper`
